Clojure Quick Reference Card

Predicates

(f= val+)
▷ true if all vals are equal via Java equals().

(fnot= val+)
▷ true if not all vals are equal via Java equals().

(fidentical? a b)
▷ true if a == b in Java, otherwise false

(f== number+)
▷ true if all numbers are equal in value.

(f< number+)
(f> number+)
(f<= number+)
(f>= number+)
▷ true if all numbers are in the order specified by the operator, else false.

(fneg? a)
(fzero? a)
(fpos? a)
▷ true if a < 0, a = 0, or a > 0 respectively.

(feven? a)
(fodd? a)
▷ true if a is even or odd, respectively.

(fnil? a)
▷ true if a is nil, otherwise false.

(ftrue? a)
▷ true if a is the value true, otherwise false.

(ftrue? a)
▷ true if a is the value false, otherwise false.

(fbigdec? a)
(fchar? a)
(fclass? a)
(fdecimal? a)
(fdelay? a)
(fdouble? a)
(ffloat? a)
(fident? a)
(fint? a)
(finteger? a)
(fkeyword? a)
(fnat-int? a)
(fnumber? a)
(fqualified-keyword? a)
(fqualified-symbol? a)
(fqualified-ident? a)
(fpos-int? a)
(fratio? a)
(frational? a)
(fsimple-keyword? a)
(fsimple-symbol? a)
(fsimple-ident? a)
(fspecial-symbol? a)
(fstring? a)
(fsymbol? a)
(fvar? a)
▷ true if a is of indicated type.

(fcoll? a)
(fseq? a)
(fvector? a)
(flist? a)
(fmap? a)
(fset? a)
▷ true if a implements standard interface for given metatype.

(fextends? protocol type)
(fsatisfies? protocal a)
▷ true if type, or type of a, extends protocol.

(finstance? javaClass a)
▷ true if a, after evaluation, is an instance of javaClass.

Numeric Functions

(f+ a+)
(f* a+)
▷ Returns Σa or Πa respectively.

(f- x y*)
▷ If no ys provided, -x, else x-Σys.

(f/ x y*)
▷ If no ys provided, 1/x, else x/Πys.

(fquot x y)
(frem x y)
▷ Returns quotient or remainder respectively of x/y.

(fmod x y)
▷ x mod y. Unlike frem, a modulus is always positive.

(finc x)
(fdec x)
▷ x+1 or x-1 respectively.

(fmin x+)
(fmax x+)
▷ Returns the lowest, or highest, x respectively.

(frand ⌞x⌝1.0)
(frand-int x)
▷ Returns a random float or int respectively, >=0 and <x.

(mwith-precision precision ⌞:rounding CEILING|FLOOR|HALF_UP|HALF_DOWN|HALF_EVEN|UP|DOWN|UNNECESSARYHALF_UP⌝ forms+)
▷ Performs BigDecimal operations in forms with the given precision and rounding method.

Structure

(sdef ⌞^{ ⌞:private true|falsefalse⌝ ⌞:doc docs⌝ ⌞:test forms⌝ ⌞:tag type⌝ }⌝ symbol ⌞docstring⌝ ⌞value⌝)
▷ Creates or locates a global var with the given name, sets it to the given value and returns it.

(mdefonce ⌞^{ ⌞:private true|falsefalse⌝ ⌞:doc docs⌝ ⌞:test forms⌝ ⌞:tag type⌝ }⌝ symbol value)
▷ Creates a global var with the given name and value and returns it, only if it does not already exist.

(fintern ns name ⌞val⌝)
▷ Creates or locates a global var named by the given name in the given namespace, sets it to val, and returns it.

(mdeclare ⌞^{ ⌞:private true|falsefalse⌝ ⌞:doc docs⌝ ⌞:test forms⌝ ⌞:tag type⌝ }⌝⎵ symbol+)
▷ Defines all of the given names, without initial values.

Sequence

(sdo form* lastform)
▷ Evaluates forms in order then lastform.

Choice

(sif test then else)
(mif-not test else then)
▷ then if test is true (non-nil, non-zero and non-false), or else otherwise.

(mif-let [var test] then ⌞else⌝)
(mif-some [var test] then ⌞else⌝)
▷ If test is true (or for mif-some only non-nil), (mlet [var test] then), otherwise else.

(mwhen test then+)
(mwhen-not test else+)
(mwhen-let [var test] then+)
(mwhen-some [var test] then+)
▷ Equivalent to sif, mif-not, mif-let and mif-some but accepting multiple forms which are wrapped in an implicit sdo.

(mwhen-first [var seq] form+)
▷ If seq is true, (sdo forms) with var bound to the first element of seq.

(mcond (test⎵value*))
▷ Evaluates each test in order; when one is true, corresponding value.

(mcondp pred expr (testexpr⎵resultexpr|textexpr⎵:>>⎵resultfn+ ⌞default⌝))
▷ Evaluates (pred testexpr expr) for each testexpr in order. When one is true, corresponding resultexpr or (resultfn (pred testexpr expr)). If none are true, default or eIllegalArgumentException.

(mcond-> expr (test⎵form+)
(mcond->> expr (test⎵form+)
▷ Using m-> or m->> respectively, threads expr through all forms where corresponding tests are true. Note that expr is not passed to any test.

Iteration

(mloop [var⎵binding*] form*)
▷ Evaluates forms with the given bound vars and serves as a marker for srecur.

(srecur binding+)
▷ Rebinds vars in the enclosing mloop to the values of bindings in order, then returns control to the start of that mloop.

(ftrampoline f args*)
▷ If the value of (f args) is not a function, (f args). Else, (ftrampoline (f args)).

(mwhile test form*)
▷ Evaluates forms until test returns false or nil.

(mfor [var⎵listexpr⎵ ⌞:let [binding⎵expr+]⌝⎵ ⌞:while expr⌝⎵ ⌞:when expr⌝+] form*)
▷ Evaluates forms with vars bound to all values in each listexpr and returns a sequence of the results.

(mdoseq [var⎵listexpr⎵ ⌞:let [binding⎵expr+]⌝⎵ ⌞:while expr⌝⎵ ⌞:when expr⌝+] form*)
▷ Evaluates forms for side effects with vars bound to all values in each listexpr and returns nil.

(mdotimes [var max] form*)
▷ Evaluates forms for side effects with var bound to each int from 0 to max minus 1.

Java Interop

(snew classname arg*)
(classname. arg*)
▷ Returns a new instance of Java/JavaScript class classname, passing args to the constructor.

(.field instance arg*)
(s. instance field|(field arg*)|field arg*)
▷ Returns the value of property, or the result of calling method, field in object instance, passing arguments args.

(.-field instance)
(s. instance -field)
▷ Returns the value of property field in object instance. Cannot call methods.

(class/field arg*)
(s. class field|(field arg*)|field arg*)
▷ Returns the value of static property, or the result of calling static method, field in class class.

class/field
▷ Returns the value of static property field in class.

(.field class arg*)
▷ Returns the value of property, or the result of calling method, field on the Class object representing class class for reflection.

(m.. instance|class field+)
▷ Nested property access. Equivalent to applying .field for each member of fields in order, starting from instance or class.

(mdoto instance (method arg*)*)
▷ Calls each method in methods - which must be in .method format - in order, with its appropriate args, on instance.

(mgen-class :name name ⌞:extends parentObject⌝ ⌞:implements [interface+]⌝ ⌞:init conName⌝ ⌞:constructors {[paramtype+]⎵[superparamtype+]+}⌝ ⌞:post-init name⌝ ⌞:methods [ [name [paramtype*] returntype]+]⌝ ⌞:main boolfalse⌝ ⌞:factory facname⌝ ⌞:state statename⌝ ⌞:exposes {fieldname⎵{:get settername :set settername}+}⌝ ⌞:exposes-methods {supermethodname⎵exposedname+}⌝ ⌞:prefix string"-"⌝⌞:impl-ns namespace⌝ ⌞:load-impl-ns loadimpltrue⌝)
▷ Creates a compiled class file named name with methods which look up Clojure functions in namespace namespace with names equal to the name of the method prefixed with prefix.

(mreify parent⎵(method [arg+] body)*+)
▷ Creates an instance of an anonymous class extending each parent and implementing or overriding each method in that parent with the code in body.
